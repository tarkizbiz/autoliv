With this app, individual user can select the timer to reload List or Kanban view without any setting in backend.





List features:

1. There are 3 timer options. You also can input another timer to reload view.


2. Auto save setting. Whenever user go back the list or kanban view, auto reload will be activate.
For example, If you select list sale order will be reloaded in 5 minutes, and you go to sale order form for details, and then if you go back list sale order. The list sale order will be reloaded in 5 minutes automatically.


3. Still keep search conditions, filters or group by after reload view (don't refresh browser). The content of view is always match with search conditions or filters.


4. Only reload list view with readonly mode. It's mean, when you are editing some things on list view, the reload is stopped automatically, it prevent to lost your data.


5. Reload will also effect when user minimize web browser or tab browser is inactive
For example, If user minimize web browser or inactive web browser, the reload is still activate.
